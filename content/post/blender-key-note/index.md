'Bar + 网格 +分离 + 选中项 ' 分离选中平面  
'X +融合' 融合平面  
'G + (Z)+ Ctrl' 吸附最近平面  
'Alt + Z'透显
 loop tools-briege 桥接循环边，开洞用

Ctrl + B 倒角 但是用于给Ctrl + R 的线给个厚度  
## 如何用线来建模
1. 新建一个带线物体  
2. 编辑模式 - 复制线   
3. p 分离选中项  
4. e 挤出  
5. 转换为曲线 - 添加倒角(给予厚度)
